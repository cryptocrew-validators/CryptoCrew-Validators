## About SEDA

SEDA is a Proof-of-Stake blockchain and modular data oracle network that enables any chain to access verifiable, programmable off-chain data. Unlike traditional oracles that push a single data feed, SEDA allows developers to write custom data request programs (Oracle Programs) that are executed in a decentralized overlay network and settled on-chain with cryptographic proofs. This makes SEDA chain-agnostic and adaptable to any data type — from price feeds to AI model outputs. SEDA is the native token used for staking, governance, and incentivizing data providers.

## How to earn rewards by staking SEDA

Stake SEDA tokens by delegating them to validators using a supported wallet. In return, you earn a share of network rewards proportional to your stake. Rewards accrue continuously and can be claimed anytime, while unstaking is subject to a 21-day unbonding period.
