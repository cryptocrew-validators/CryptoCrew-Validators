## About Fuel

Fuel is a modular execution layer and Ethereum L2 built around parallel transaction processing using a UTXO-based architecture and the custom FuelVM. Unlike EVM chains that execute transactions sequentially, Fuel processes them in parallel, enabling significantly higher throughput. Developers use the Sway programming language to write smart contracts for FuelVM. Network sequencing is secured by a Proof-of-Stake Shared Sequencer Network using Tendermint consensus. FUEL is the native token used for staking, governance, and network fees.

## How to earn rewards by staking FUEL

Stake FUEL tokens via the [Fuel Ignition staking dashboard](https://app.fuel.network/staking/on-fuel). Staked tokens accumulate FUEL rewards continuously, which can be claimed anytime. Withdrawals return to your wallet with no explicit unbonding period.
